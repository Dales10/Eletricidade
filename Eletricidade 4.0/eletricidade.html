Como usar

Para encontrar o dado que deseja calcular, primeiro escolha qual das fórmulas irá usar, e em seguida os campos onde será colocado os dados irão aparecer.

Logo após insera e escolha os dados requisitados para que possa ser calculado determinado valor (certifique-se que tem todos os dados que são necessários para que o calculo seja realizado). A primeira coluna faz referência aos primeiros dados que precisam ser inseridos, enquanto a do lado são os dados secundários ou complementares.

Sendo mais direto, alguns valores podem vir acompanhados de uma potência de base 10, e é nessa segunda coluna onde deve colocá-los, ou escolher o valor correspondente, mas só precisa inserir o número que a potência de base 10 está elevado. Ex.: 105, coloque apenas o 5 no campo.

Ademais, ao colocar os valores e eles atenderem ao critério de calculo de alguma incognita, o mesmo será feito e mostrado de imediato, sem a necessidade de apertar nenhum botão para confirmar.

Obs.: Apenas use o recurso de cálculo das 8 fórmulas presente nesse site para fins de estudo.
